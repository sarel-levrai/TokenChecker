# TokenChecker
A simple python discord token checker
## What is TokenChecker?
TokenChecker is a python tool that check if a token is valid or dead.
## How to put Tokens ?
in the `tokens.txt` file, you can put tokens in this format :
```
token1
token2
token3
```
## Dependencies
run `setup.bat`
run `index.py`
enjoy!
